## Design Characteristics

- **Theme**: Dark mode primary (with light mode support)
- **Color Palette**:
  - Primary: #137FEC (blue)
  - Background: #101922 (dark)
  - Surface: #1A2632 (dark)
  - Keys: #233648 (dark)
- **Typography**: Space Grotesk (Google Fonts)
- **Icons**: Material Symbols
- **Layout**: Mobile-first, 4-column grid
- **Effects**: Glassmorphism, smooth animations

## Usage

These files serve as the visual reference for implementing the calculator UI. The design should be followed closely to maintain consistency and premium aesthetics.
